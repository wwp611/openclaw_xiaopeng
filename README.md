## 🎯 核心功能
### 对称旋转算法
基于用户输入的8×8图案，自动生成16×16对称图案：

```
✅ 正确四象限布局：
┌────────────┬────────────┐
│ 第二象限    │ 第一象限    │
│ (左上)     │ (右上)     │
│ 旋转90°    │ 原始输入    │
├────────────┼────────────┤
│ 第三象限    │ 第四象限    │
│ (左下)     │ (右下)     │
│ 旋转180°   │ 旋转270°   │
└────────────┴────────────┘
```

## 🚀 功能特性
- ✅ **对称旋转算法** - 8×8输入 → 16×16输出
- ✅ **四象限实时预览** - 同时显示四个旋转版本
- ✅ **移动端优化** - 专为手机触摸设计
- ✅ **多种输出格式** - 16×16矩阵、一维数组(256元素)
- ✅ **一键复制** - 方便粘贴到代码中
- ✅ **算法验证工具** - 支持L形等测试图案
- ✅ **GitHub Pages托管** - 永久免费公网访问

## 📱 使用流程
1. **输入**：在8×8网格中选择格子（第一象限）
2. **生成**：点击"生成对称图案"按钮
3. **查看**：
   - 四个象限分别显示旋转结果
   - 16×16最终图案可视化
   - 矩阵和一维数组输出
4. **复制**：使用复制按钮获取格式化结果
