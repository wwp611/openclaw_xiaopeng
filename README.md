## 🆕 新增功能：8×8网格选择器

### 📋 功能描述
已成功创建交互式8×8网格选择器网页应用，包含以下功能：

#### 核心特性
- 🎯 **8×8交互式网格**：可视化点击选择界面
- 🔄 **双向切换**：点击选择，再次点击取消选择
- 📊 **实时统计**：显示已选中的格子数量
- 🔢 **二进制矩阵输出**：自动生成8×8矩阵（1=选中，0=未选中）

#### 操作控制
- ✅ **生成矩阵**：将选择结果转换为二进制矩阵
- 🔄 **清除选择**：一键重置所有选择
- 🌟 **全选功能**：快速选择所有64个格子
- 📋 **复制矩阵**：一键复制矩阵到剪贴板

#### 输出格式
1. **二维矩阵格式**（8×8网格展示）
2. **一维数组格式**（Python/JavaScript兼容）
3. **CSV格式**（电子表格兼容）

### 🚀 快速使用
1. 访问 `grid-selector.html` 文件
2. 在8×8网格中点击选择需要的格子
3. 点击【生成二进制矩阵】获取结果
4. 点击【复制矩阵】将结果复制到剪贴板
